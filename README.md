# yoocAutoGenerator

Yooc学术交流、分组讨论

## 使用方式

在浏览器 /WebView/Electron 中浏览 index.html 文件。

TO-DO:

你也可以在 Node.js 中使用 index.js 作为模块。

## 作用效果

将互联网上的题目粘贴到题目栏中，即可自动提取题目和选项数据并生成特定格式的题目文件。目前支持单选题、多选题。

TO-DO:

支持判断题、简答题

TO-DO:

你也可以使用多人协作功能查重和处理协作题目，并自动合并。

你可以自动生成非常棒的题目解析。形如由于……所以……这样的格式。
